# E-Commerce Sales Analytics Dashboard

An interactive Power BI dashboard built to analyze e-commerce sales performance and generate actionable business insights. This project focuses on sales trends, customer behavior, product performance, and regional analysis using data visualization and KPI tracking.

## Features

* Sales Performance Analysis
* Profit and Revenue Tracking
* Monthly Sales Trend Analysis
* Top 10 Products by Sales
* Category-wise Sales Distribution
* Region-wise and State-wise Sales Analysis
* Customer Segment Analysis
* Interactive Filters and Slicers
* Executive Key Insights Dashboard

## Tools & Technologies

* Power BI
* Microsoft Excel
* SQL
* DAX (Data Analysis Expressions)

## Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Average Order Value (AOV)
* Profit Margin %

## Business Insights

The dashboard helps identify top-performing products, high-revenue regions, customer purchasing patterns, and seasonal sales trends, enabling data-driven business decisions.
